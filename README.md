# INFOMITRA 🇮🇳

INFOMITRA is an AI-powered web platform that helps citizens understand government documents and schemes in a simple and user-friendly way.

The platform provides step-by-step guidance, eligibility information, required documents, and simplified explanations for various government services.

---

# ✨ Features

- 📄 Government Document Guidance
- 🎗️ Government Scheme Information
- 🤖 AI-powered Search using Gemini API
- 🌐 Multilingual Support
- 📂 Document Upload Feature
- 🛣️ Next Step Guidance System
- 📱 Responsive UI Design
- ⚡ Fast Search Experience

---

# 🛠️ Tech Stack

## Frontend
- React.js

- CSS
- Lucide React Icons

## Backend
- Node.js
- Express.js

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone <your-repository-link>
```

---

## 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

## 3️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```

---

# 🔑 Environment Variables

Create a `.env` file inside backend folder.

```env
MONGO_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_gemini_api_key
PORT=5000
```
